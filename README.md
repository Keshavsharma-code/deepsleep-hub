# deepsleep-hub
Browser extension that unifies ChatGPT, Claude, Gemini, Kimi (and 6+ others) into a living, pulsing 3D knowledge graph. Real-time semantic extraction, vector embeddings, and 2KB memory compression—visualized as ink-splatted neural lobes that glow, crack, and overflow when you hit cognitive limits. Your second brain, with chromatic aberration.

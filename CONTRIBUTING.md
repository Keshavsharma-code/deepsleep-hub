## 📥 Getting Started

1. **Fork & Clone**:
   ```bash
   git clone https://github.com/Keshavsharma-code/deepsleep-hub.git
   ```
2. **Setup**:
   - Open Chrome Extension Management (`chrome://extensions`).
   - Enable "Developer mode".
   - Click "Load unpacked" and select the `deepsleep-hub` folder.
3. **Build**:
   - No build step required for core logic (Vanilla JS).
   - Edit, Save, and Refresh.

## 📜 Principles
- **Aesthetic First**: If it looks generic, it's not DeepSleep. Use gradients, noise, and glow.
- **Privacy First**: All data must stay 100% local. No cloud syncing without explicit permission.
- **Performance Matters**: Keep the UI fluid at 60FPS.
